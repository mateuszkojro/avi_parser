# Collection of scripts and programs for DIMS project

## How to run?

Every subproject has an `run.sh` and optionaly `test.sh` scripts making it easy

## Dependencies

### C++

1. CMAKE
1. make
1. some recent c++ compiler

_For running OpenCL_

1. OpenCL headers

### Python

_Normal_

1. Python3
1. Numpy
1. Matplotlib
1. Jupyter Notebook

_Testing_

1. pytest
1. pylint
